# BIG-BANG Script (FizzBuzz Variation)

This TypeScript script prints numbers from 1 to 100, replacing:
- Multiples of 3 with `"BIG"`
- Multiples of 5 with `"BANG"`
- Multiples of both 3 and 5 with `"BIGBANG"`
